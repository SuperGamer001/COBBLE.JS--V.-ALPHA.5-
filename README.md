# Sonic Blocked!

This game is a work in progress using an engine I'm activly building called Cobble.js.

In this fan-made Sonic game, you play as Sonic in a blocky state (Inspiration by Gamemode One's Minecraft: Sonic the Hedgehog map). He can walk -> run, jump, and even tap his foot when idle for too long.

### Can You Collect All 35 Rings On The Tiny Alpha Stage?

## How to Install
First you will need three things for this to work:
- VS Code (https://code.visualstudio.com/download)
- Live Server Extention for VS Code (https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
- Node.js (https://nodejs.org/en)

Go into VS Code and open the folder where you want to store the game. (File -> Open Folder [ctrl + K + ctrl + O])

Open the panel (ctrl + J) and go to the Terminal tab. Enter the following code to install everything you need:

```
git clone https://github.com/SuperGamer001/COBBLE.JS--V.-ALPHA.5-.git
cd COBBLE.JS--V.-ALPHA.5-
npm install three
```

## How to play
After you have installed everything you need, open the new folder that your terminal created with VS Code.
On the bottom right corner, if you installed the Live Server Extention, you should see a "Go Live" button. Press that to start a local server.

### Controls:
- Walk & Run: W
- Rotate Camera: A and D
- Hit the breaks: S
- Jump: Space
- Tap Foot: Do Absolutly Nothing
